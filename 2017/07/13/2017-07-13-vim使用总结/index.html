Vim使用总结

文章作者：Tyan
博客：noahsnail.com  |  CSDN  |  简书

本文主要是记录使用vim过程中的一些用法，本文中介绍的命令都在vim中使用验证过。

1. 删除一行或多行

  • 删除一行，命令格式：[:行号d]
  • 删除多行，命令格式：[:起始行号,结束行号d]

2. 显示/不显示行号

  • 显示行号
  • 不显示行号

3. 字符串替换

  • 命令格式：[:%s/原始字符串/要替换的字符串]

4. 跳到指定行

  • 命令格式：[:line-number]

5. 防止粘贴自动缩进

  • 命令格式：[:set paste]

6. 粘贴后允许自动缩进

  • 命令格式：[:set nopaste]

7. 取消选中或高亮

8. 撤销上一步操作

按u键即可。

9. 纵向编辑

  • Ctrl + v进行纵向模式

  • 按上下键选择要编辑的区域

  • 按shift + i进入编辑模式

  • 按Esc键整体编辑结束
